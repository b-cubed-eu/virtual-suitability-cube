# vscube

<!-- badges: start -->
<!-- badges: end -->

vscube è un pacchetto R per costruire un virtual suitability cube:

scarica variabili climatiche mensili da WorldClim

-   legge e pulisce occorrenze GBIF

-   allena modelli di idoneità (SDM) con MaxEnt (enmSdmX)

-   predice in nuove aree e aggrega i risultati in formato stars
